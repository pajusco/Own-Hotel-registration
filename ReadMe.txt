  * *  -- Hotel Registration App -- * * 

    Programme implementation : 

  - Booking a room and providing information such as room number, person name who is booking, number of people in the room
  - Display of room information ( all rooms at once )  such as number of persons in room, name of the person making the reservation, room status.
  - Display of room information ( using the room number ) such as number of persons, name of the person making the reservation.
  - Checking the number of available rooms.
  - Checking the status of the room ,using the room number  ( Free / Busy ).
  - Display of average number of people per room.
